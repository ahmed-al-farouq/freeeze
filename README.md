    - ReactJs

### Setup
  * Open your command
  * Type "git clone https://github.com/ahmed-al-farouq/Metrics-Webapp.git"
  * npm i
  * npm run start
### Run the app
  * npm run start

### Deployment
  * npm run build

## Testing
